# 🏥 Hospital Management System

A Java Swing + MySQL based desktop application to manage hospital operations, including authentication, patient registration, room assignment, and ambulance tracking.

---

## 🔧 Technologies Used
- **Programming Language**: Java
- **GUI Framework**: Java Swing
- **Database**: MySQL
- **IDE**: IntelliJ IDEA

---

## 🚀 Features
- 🔐 **Login Authentication** for admin and staff
- 🧾 **Patient Registration** with real-time data validation
- 🏠 **Room Assignment System**
- 🚑 **Ambulance Tracking**
- 📊 User-friendly desktop interface with error handling and UI feedback

---

## 📁 Project Structure
├── src/
│ ├── login/
│ ├── registration/
│ ├── rooms/
│ ├── ambulance/
│ └── db/
└── HospitalManagementSystem.iml
## 🛠 How to Run
1. Clone the repository:
git clone https://github.com/shambhavi0608/Hospital-Management-System--by-Shambhavi.git

2. Open in **IntelliJ IDEA**
3. Setup MySQL Database with provided SQL schema
4. Update DB credentials in `DBConnection.java`
5. Run the `Main.java` class
